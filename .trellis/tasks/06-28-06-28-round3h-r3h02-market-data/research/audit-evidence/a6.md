## 计划外 perf 扫描（Plan 外 · 只读 · ports 范围）

> 未跑 benchmark；静态审阅五源 port + normalizer，评估若将来接 live 时的 obvious 风险。

| 检查项       | 观察                                                                       | 风险                |
| ------------ | -------------------------------------------------------------------------- | ------------------- |
| 默认路径 I/O | 五源 MockFetchPort：**零网络**；yahoo 可选读本地 replay JSON（单文件）     | 低                  |
| payload 大小 | mock bar/surface ≤500 行 cap；`json.dumps` 小对象                          | 低                  |
| 循环复杂度   | bar 生成 O(min(max_rows, 3))；Deribit surface O(1) mock                    | 低                  |
| 重复磁盘读   | `yahoo_finance_port._mock_bars` 每次 fetch 可 `read_text` replay（L49–50） | 低（mock 测试频率） |
| Live 预留    | AV `AlphaVantageLiveFetchPort` 仍委托 mock（`ponytail:` L148）             | 无 live perf 债务   |
| Normalizer   | read/write 全内存 JSON；无 DuckDB/Parquet 扫描                             | 低                  |
| 并发         | 无 shared mutable state；dataclass frozen port                             | 低                  |

**结论：** 当前实现无 hot path；live wiring 切片（urllib/websocket）未落地，**不适用** SLA/RSS 门禁。若 Batch 3H 后续接 live，建议在 **独立切片** 补 rate-limit + timeout + 窗口 cap 压测，而非本卡 A6 范围。
