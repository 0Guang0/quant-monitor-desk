## 信任边界（契约 + 任务卡 Red Flags）

### 无 live fetch / 无 production 写（playbook §2.1 / §3.2）

- `EvidenceChainBuilder`、`StagedEvidenceValidator`、`InstrumentRegistryValidator` **纯内存** 返回 dataclass；无 DuckDB 连接、无 `writer()`、无 migration。
- 与 `docs/modules/layer5_security_evidence.md` §1「不绕过 WriteManager 写入」一致；当前 023b 切片**未引入写路径**，无 bypass。
- Wave D Track B 边界：`BATCH_01_MAIN_SESSION_COORDINATOR_PLAYBOOK.md` §2.1 禁止 live source、production 写入；diff 未触碰 Batch 01 核心路径。

### Agent 文本非事实源（AC-023-3 / 任务卡 §8 Red Flags）

- `EvidenceFoundationValidator.reject_agent_text_as_fact_source`：`created_by` 以 `agent` 开头或摘要含 `agent_summary`/`建议`/`买入`/`卖出`/`信号` → `EvidenceFoundationError`（`foundation.py:47-55`）。
- `EvidenceChainBuilder.build` 在 `FACTUAL_SOURCE` 路径委托 foundation 校验（`evidence_chain.py:45`）。
- 测试：`test_evidenceChain_rejectsAgentTextAsFact` + 023A `test_evidenceFoundation_agentTextNotFactSource_rejectsFactualKind`。

### severe conflict → manual review queued（ADR-023 / AC-023-4）

- `conflict_severity == "severe"` → `need_human_review=True` + `manual_review_state=queued`（`evidence_chain.py:96-97`）。
- 不一致态 `queued` 且无 `need_human_review` → `EvidenceChainError`（`evidence_chain.py:98-101`）；与 023A `_validate_manual_review` 对齐。
- 测试：`test_evidenceChain_severeConflictQueuesManualReview`。

### as_of fail-closed（no-future-data）

- `StagedEvidenceValidator.validate_bar`：`trade_date > as_of_end` → `EvidenceValidationError`（`evidence_validator.py:27-30`）。
- 测试：`test_securityBar_rejectsFutureTradeDate`。

### EvidenceReadPort 注入边界（R2-RISK-2 / AC-023-5）

- `ports.py` 仅定义 `Protocol`；**无** concrete storage 实现 import。
- `build_from_port` 无 port 时 fail-closed（`evidence_chain.py:73-74`）。
- 测试 `_FakeEvidenceReadPort` 内存 fake；`test_evidenceReadPort_boundary` 冻结解耦语义。
- **安全收益：** 当前无文件路径解析面；未来 concrete port 须单独过路径守卫审计。

### upstream snapshot 与 layer context 必填

- 空 `upstream_snapshot_ids` → `EvidenceChainError`（`evidence_chain.py:49-50`）。
- 空 `layer3_context` / `layer4_context`（strip 后）→ `EvidenceChainError`（`evidence_chain.py:51-52`）。
- 正向测试：`test_evidenceChain_traceUpstreamSnapshots`；**否定路径见计划外发现 OOB-A3-1**。

### 血缘 / Agent 污染面（023A 延续）

- `Layer5LineageBuilder` 复用 `validate_source_dataset_ids` + `assert_lineage_fields_complete`（`lineage.py:86-114`）。
- `agent_interpretation` 禁止携带 factual provenance（`foundation.py:90-95`）。

### 交易动作语义阻断

- `foundation.py` `_AGENT_TEXT_PATTERN` 含 `买入|卖出|信号`；契约 `no_trading_action_semantics: true`。
- builder 未输出买卖建议字段；仅 staged 证据摘要槽。
