# 环境轴（ENVIRONMENT）用户说明

## 一句话定义
环境轴看的是宏观世界里的“水位、利率、通胀和实体经济底板”。

它回答：钱的水位高不高？无风险利率在什么位置？通胀压力在哪里？实体经济有没有变弱？

## 用户怎么理解这条轴
可以把环境轴理解成金融市场的“天气和水位表”。它不直接告诉你买什么卖什么，而是告诉你市场所处的大环境是否偏紧、偏松、偏热、偏冷。

## 它能说明什么
- 央行资产负债表、财政账户、现金流通、逆回购等因素是否在抽水或放水。
- 短端利率和长端利率处于什么水位。
- 收益率曲线是否倒挂或变陡。
- 真实利率和期限溢价是否显示紧缩压力。
- 通胀预期和实际通胀是否偏热。
- 就业和失业是否显示经济底板变弱。

## 它不能说明什么
- 不说明企业信用压力。信用利差、融资楔子归信用压力轴。
- 不说明期权市场愿意为尾部风险付多少钱。VIX、SKEW、VRP 归风险偏好轴。
- 不说明交易成本和盘口摩擦。价差、冲击成本归流动性轴。
- 不说明仓位拥挤或情绪。COT、PCR、AAII、NAAIM 归情绪轴。

## 用户版前端展示建议
每个指标展示：

- 今日值
- 历史位置：高位 / 低位 / 正常 / 极端
- 今日变化：上升 / 下降 / 平稳 / 突变
- 数据滞后：几天
- 数据质量：正常 / 滞后 / 缺失 / 历史不足 / 源异常
- 通俗解释：今天这个位置意味着什么
- 边界提醒：这个指标不能解释什么

## 典型解释示例

### 准备金余额 WRESBAL
通俗解释：银行体系里的“水池存量”。数值高说明银行体系中可用准备金较多，数值低说明水池变浅。

注意：它告诉你水池里有多少水，但不直接告诉你是谁在抽水或放水，需要配合净水位 proxy 的四条腿解释。

### 净水位诊断线 NET_LIQ_PROXY_BN
通俗解释：把 Fed 总资产、财政账户、流通中货币和隔夜逆回购拼起来，帮助判断到底是哪条腿在抽水或放水。

注意：这是混频投影，不是严格日频真值，只能用于归因解释，不参与核心计分。

### EFFR / DGS10 / T10Y3M
通俗解释：分别看短端政策坐标、长端名义利率水位和收益率曲线形态。

注意：曲线倒挂只说明期限结构形态，不直接解释倒挂原因。

### DFII10 / ACMTP10
通俗解释：DFII10 看真实利率压力；ACMTP10 看长期期限风险补偿。

注意：ACM 是模型输出且有滞后，不能伪装成实时日频状态。

### CPI / T10YIE / T5YIFR
通俗解释：CPI 看已经发生的通胀温度；T10YIE 和 T5YIFR 看市场定价中的通胀锚。

注意：CPI 是月频滞后数据，不用于日内判断。

### 失业金 / 失业率 / Sahm Rule
通俗解释：看实体经济有没有失血、滞留或衰退体征。

注意：周频失业金容易受节假日、灾害、罢工影响，单周值只能做解释灯，主状态看 4 周均值。

## 应保留的信息
- 原始数值、单位、来源、发布时间、抓取时间。
- 历史百分位、z-score、robust z-score、变化量。
- 数据滞后与 stale_reason。
- 是否为诊断项、背景项或盲区。

## 可从用户版删除的信息
- Stage I / Stage II 验收过程。
- 每个端点的探针日志。
- 详细 schema_hash、content_fingerprint 技术说明。
- Shadow 端点长说明。
- 重复的 no action semantics 文字。
