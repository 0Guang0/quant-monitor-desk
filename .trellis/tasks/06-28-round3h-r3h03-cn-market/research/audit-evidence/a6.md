## 2. SKIP 理由（§3.6 等价）

### 2.1 AUDIT.plan 冻结理由（复述 + 证实）

1. **CN 低频 batch/replay** — frozen §1.1「replay-first；live 仅用户显式授权子集」；九源 mock/replay 为默认路径，不经 orchestrator 高频轮询。
2. **无生产 hot path / SLA** — 十源 port 由 pytest、`layer_cn` smoke、route planner 测试驱动；不经 FastAPI 用户面、不经 `SyncOrchestrator` 默认定时拉取、不经 `production_equivalent_smoke.py`。
3. **禁止主库写与全市场扫描** — frozen §2.8/§8；`GLOBAL_RESOURCE_LIMITS.md` §3 禁止默认全市场全历史；实现层 `FULL_MARKET_SCAN_ENABLED = False`（tdx）、symbol whitelist（baostock）。
4. **无 Web / CWV 面** — 本任务 diff 为 backend fetch port + registry；LCP/INP/CLS 不适用。
5. **Plan 无 perf 冻结行** — A8 子集 pytest 为功能门禁；无时长/内存数字可供 A6 PASS/FAIL。
