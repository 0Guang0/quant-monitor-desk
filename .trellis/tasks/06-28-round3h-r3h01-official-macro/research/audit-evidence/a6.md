## 1. SKIP 理由（AUDIT.plan §1 复述 + 证实）

1. **低频官方披露 API** — FRED / Treasury / SEC / CFTC / BIS / World Bank 均为日/周/季频宏观与披露源；frozen §8 禁止 minute-level feed、禁止 full-history macro pull。
2. **无生产 hot path SLA** — 六源 `create_*_fetch_port` 仅被 pytest 与 `fred_sandbox_pilot`（R3E 旁路）引用；**未**注册进 `create_adapter` / `DataSourceService` 主 fetch 链；不经 `SyncOrchestrator` 或 Web API。
3. **mock-first / live 推迟** — 除 FRED opt-in live 外，Treasury / SEC / CFTC / BIS / World Bank 的 live 路径在 R3H-01 显式 `PortError` 或 identity gate only。
4. **caps 在 port 层硬编码** — 各源 `MAX_*` + 白名单 + `_reject_over_cap`；与 frozen §7 / `source_capabilities.yaml` 对齐（见 §2）。
5. **Tier A 无 perf 冻结行** — `EXECUTION_INDEX.md` §2 通过条件 = pytest exit 0；无 smoke 时长/内存阈值供 A6 PASS/FAIL。

## 4. 计划外 perf 风险扫描

> AUDIT.plan A6 虽 SKIP，仍须扫描：无界 I/O、O(n²)、新 port 缺 cap。

### 4.1 Hot path

| 路径                                                      | Hot?     | 证据                                              |
| --------------------------------------------------------- | -------- | ------------------------------------------------- |
| 六源 `create_*_fetch_port` → `fetch_payload`              | **否**   | 无 `create_adapter` 注册；无调度/HTTP 入口        |
| `fred_sandbox_pilot.run_mock_fetch` / `run_live_fetch`    | **否**   | R3E 旁路；caps + 串行 series 循环                 |
| `live_pilot_phase3.capture_phase3_raw_evidence`           | **邻接** | 仅 FRED；非本卡新增五源                           |
| `ingestion_evidence.official_macro_bundle_layer1_preview` | **否**   | O(n) `len(observations)`；smoke 用 replay fixture |

### 4.2 无界 I/O

| 面                | 扫描结果                                                            | 评级                                              |
| ----------------- | ------------------------------------------------------------------- | ------------------------------------------------- |
| Live HTTP         | 仅 `FredLiveFetchPort`：`urlopen(timeout=30)` + API `limit` + slice | **有界**；`response.read()` 全量读后 slice — NB-5 |
| 其他五源 live     | 显式 disabled 或 mock-only                                          | **无网络**                                        |
| 证据文件读        | `read_*_evidence_bundle`：`Path.read_text` 单次 JSON                | **依赖 fixture 体量**；replay bundle 小           |
| DuckDB / Parquet  | 六源 port 零引用                                                    | **无**                                            |
| full catalog 扫描 | 各源 `frozenset` 白名单                                             | **无**                                            |

### 4.3 O(n²) / 批大小 / 内存

| 面                                    | 行为                                  | 风险                                                    |
| ------------------------------------- | ------------------------------------- | ------------------------------------------------------- |
| `fred_observations_from_live_payload` | 双层 `for series / for row`           | O(series×rows)；pilot ≤5×100，L2 ≤10×500 — **线性有界** |
| normalizer `build_*`                  | 列表推导 `_normalize_*`               | O(rows)；≤500 — **可忽略**                              |
| `json.dumps` bundle                   | 全 bundle 入内存                      | ≤500 obs × 6 源独立调用 — **可忽略**                    |
| `fred_sandbox_pilot` 多 series 串行   | `for series_id in request.series_ids` | 无并行风暴；network_calls cap                           |

**结论：** 无 O(n²) 阻断模式；声明 caps 下内存/CPU 可忽略。
